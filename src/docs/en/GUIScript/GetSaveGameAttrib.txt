
Prototype: GemRB.GetSaveGameAttrib(Type, SlotCount)

Description: Returns the name, path, prefix and elapsed game time of the
passed saved game.

Parameters:
Type - the queried attribute's name
 0 - name
 1 - prefix
 2 - path
 3 - date
 4 - elapsed game time
 5 - id
SlotCount - the index of a saved game

Return value: string
